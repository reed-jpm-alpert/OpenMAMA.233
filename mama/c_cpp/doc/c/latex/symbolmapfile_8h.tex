\label{symbolmapfile_8h_abe827f43cd4fa4bac26da768d36a79b1}


Create a file-\/based symbol mapper. Use \hyperlink{symbolmapfile_8h_a8cc5fda99d5bfb16091e4099f8ac97bb}{mamaSymbolMapFile\_\-load()} to open the file. If the file exists, its contents should be a two columns of data, with the columns separated by white space. A matching symbol of the left column is mapped to the symbol in the right column. when performing the mapping, if the symbol does not match anything in the file (or the file cannot be found), then the original symbol is used (i.e., no mapping).
