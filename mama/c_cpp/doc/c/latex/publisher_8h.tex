\subsubsection[{void}]{\setlength{\rightskip}{0pt plus 5cm}typedef void (MAMACALLTYPE $\ast$ {\em mamaThrottledSendCompleteCb})}}
\label{publisher_8h_aa3f69bd868408c3133d6cda13fa3ac56}


Responsible for publishing mama messages. Prototype for callback invoked when a message sent with either \hyperlink{publisher_8h_a04d389414b998df0518cfd46b53a45d7}{mamaPublisher\_\-sendWithThrottle()} or \hyperlink{publisher_8h_a8ffbe52cb9e0c386950184f53cdbc755}{mamaPublisher\_\-sendFromInboxWithThrottle()} has been sent from the throttle queue.

Messages sent on the throttle queue are no longer destroyed by the API. It is the responsibility of the application developer to manage the lifecycle of any messages sent on the throttle.


\begin{DoxyParams}{Parameters}
\item[{\em publisher}]The publisher object used to send the message. \item[{\em msg}]The mamaMsg which has been sent from the throttle queue. \item[{\em status}]Whether the message was successfully sent from the throttle. A value of MAMA\_\-STATUS\_\-OK indicates that the send was successful. \item[{\em closure}]User supplied context data. \end{DoxyParams}
